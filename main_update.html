About

This website serves as an updated version of CSE383 Web Application Programming final project. The objective was to create a responsive, degradable webpage with a menu that leads to a cloud API calculator and an API weather forecast. I wasn't 100% happy with my resulting final, as I had failed to make my site responsive and degradable. Therefore, I decided to work on page responsiveness and degradability, and I have updated the page CSS to match the overall clean and more modern look of my website.
